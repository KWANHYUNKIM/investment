# 음식료 업종 원가구조 분석 (파일럿)

한국 상장사를 **상향식(bottom-up) 원가구조**로 분석하는 프로젝트의 첫 클러스터.
"무엇을 팔고 → 원재료가 뭐고 → 얼마나 올랐고 → 얼마가 남는지"를 회사별로 정리한다.

## 파일럿 5사 비교표 (최신 회계연도 기준)

| 회사 | 매출 | 영업이익률 | 매출원가율 | 원가 핵심 | 전가력 | 마진 방향 |
|---|---|---|---|---|---|---|
| [오리온](오리온_271560.md) | 3.33조 (+7.3%) | **16.7%** ⭐ | ~60% | 팜유·소맥, 코코아↓ | ★★★★ | 코코아 급락 수혜 |
| [농심](농심_004370.md) | 3.51조 (+2.2%) | 5.2% | ~71% | 소맥↓·팜유↑ (원가 80%) | ★★★ | 곡물↓+해외성장 |
| [CJ제일제당](CJ제일제당_097950.md) | 17.87조 (+—) | 5.8% | ~69% | 곡물↓ / **라이신 판가↓** | 식품★★★★ 바이오★★ | 바이오 스팟가에 좌우 |
| [오뚜기](오뚜기_007310.md) | 3.54조 (+2.4%) | 6.3% | **82.9%** | 팜유↑·대두유↓·환율 | ★★★ | 유지+환율 이중노출 |
| [롯데칠성](롯데칠성_005300.md) | 3.97조 (-1.3%) | 4.2% | ~64% | **포장재(캔+PET) 50%** | ★★★ | 나프타·환율 민감 |

### 읽는 법 (투자 함의)
- **전가력 × 마진 방향**이 핵심. 오리온은 강한 브랜드(★★★★) + 코코아 급락으로 가장 유리한 구도.
- 원가 비중이 높을수록(농심 80%, 오뚜기 83%) 원재료 하락 시 **마진 레버리지**가 크다.
- CJ제일제당은 식품이 아니라 **바이오(라이신) 스팟가격**이 실적 스윙 요인 — "음식료"로만 보면 오판.
- 롯데칠성은 유일하게 **포장재가 최대 원가** — 곡물이 아니라 알루미늄·나프타를 봐야 함.

## 음식료 업종 공통 원자재 바구니 & 가격 소스
이 매핑을 한 번 만들어두면 업종 내 수십 개 회사에 재사용 가능(확장 핵심).

| 원자재 | 영향받는 회사 | 가격 소스 |
|---|---|---|
| 소맥(밀) | 농심·오뚜기·오리온·CJ | CBOT 소맥선물, 통계청 PPI |
| 팜유·대두유 | 농심·오뚜기·오리온 | 말레이시아 팜유(MDEX), CBOT 대두유 |
| 원당·설탕 | CJ·롯데칠성·오뚜기·오리온 | 뉴욕 원당선물(ICE) |
| 옥수수 | CJ(전분당·발효) | CBOT 옥수수선물 |
| 코코아 | 오리온·롯데웰푸드 | ICE 코코아선물 |
| 대두 | CJ | CBOT 대두선물 |
| 알루미늄·나프타 | 롯데칠성(포장재) | LME 알루미늄, 석유공사 페트로넷 |
| 라이신·트립토판 | CJ(바이오) | 유럽·중국 스팟(증권사 리포트) |
| 원/달러 환율 | 수입원재료 전 회사 | 한국은행 ECOS |

## 방법론 (스키마)
각 회사 MD = ① 무엇을 파는가(매출비중) ② 원재료&가격추이(전가력 별점) ③ 원가구조(원가율·OPM·인건비) ④ 마진 민감도 결론. 데이터 소스는 DART 사업보고서 「II.사업의 내용 → 2.주요 제품 / 3.원재료 및 생산설비」 + 손익계산서 + 증권사 리포트.
