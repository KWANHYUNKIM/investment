# 오뚜기 (007310) | 음식료 – 종합식품(가공식품)

> 기준: **FY2024 감사 사업보고서**(2025.3.18 공시, KIND 원문 직접 추출). 원재료 방향은 FY2024 vs FY2023.

## 3. 원가구조
- 매출원가율 **약 82.9%** (5사 중 가장 높음) / 영업이익률 **약 6.3%**
- 영업이익 **2,220억 (-12.9%)** — 곡물·대두유 하락에도 팜유·설탕↑ + 고환율 + 농수산가공품 매입 확대가 상쇄
- 인건비: 직원 ~3,460명, 평균 4,900만원 → 급여 ~1,700억, 매출대비 ~6%(복리후생 포함 8~9%)

## 4. 마진 민감도 결론
- 원가가 **유지류(팜유·대두유)+소맥분**에 크게 연동. FY2024엔 소맥·대두유 하락에도 팜유·설탕·고환율이 상쇄 → 원가율 83%로 상승, OPM 7.4%→6.3%.
- 유지류가 **USD 수입품**이라 유가·팜유 국제가 + 환율에 이중 노출. 영업이익률 6%대로 얇아 **팜유 10%↑ 또는 원화 약세 시 마진 압박 즉각적**.
- 반대로 곡물·유지 동반 하락 + 원화 강세 국면에선 마진 개선 탄력 큼.
