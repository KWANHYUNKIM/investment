# 🔬 제품 단위 원가분해 — 농심 신라면 (120g 1봉지)

> **이게 "얼마에 팔아서 얼마 남는지"의 실제 깊이.** 회사 전체 원가율(71%)이 아니라 **제품 1개를 팔면 누가 얼마를 가져가는지**를 분해한다.
> ⚠️ 기업은 SKU별 원가를 공시하지 않음 → 아래는 **① 공시 원가율 ② 업계 원가구조 추정치 ③ 국제 원자재 시세**를 결합해 재구성한 **추정 모델**(가정 명시). 정밀화는 DART 제조원가명세서 + IR 문의 필요.

## 💰 소비자가 1,000원의 여정 (대형마트 기준)

```
소비자 지갑 1,000원
│
├─ 🏪 유통 마진 (도매+소매)     ~200원 (20%)  ← 농심이 못 가져감
│
└─ 🏭 농심 출고가(= 농심 매출)  ~800원 (100%)
        │
        ├─ 밀가루(소맥분)        ~150원 (19%)  ▼최근 하락
        ├─ 팜유(면 튀김)         ~150원 (19%)  ▲최근 강세
        ├─ 📦 포장재(봉지)        ~175원 (22%)  ▲"라면은 봉지값이 원가의 1/4"
        ├─ 스프·건더기(30~40종)   ~100원 (13%)  ~
        ├─ 물류·마케팅·판촉      ~135원 (17%)
        ├─ 제조노무·감가·에너지   ~48원  (6%)   ← 라인 자동화로 낮음
        │
        └─ 💵 영업이익           ~42원  (5.2%)  ← 봉지당 실제로 남는 돈
```

## 📊 핵심 수치

| 구분 | 금액(추정) | 비고 |
|---|---|---|
| 소비자가(마트) | 1,000원 | 편의점은 ~1,500원 |
| 유통이 가져가는 몫 | ~200원 | 농심 매출 아님 |
| **농심 출고가(매출)** | **~800원** | 원가율 계산 기준 |
| 제조원가(COGS) | ~570원 (71%) | 원재료+노무+경비 |
| 판관비 | ~190원 (24%) | 물류·마케팅 |
| **봉지당 영업이익** | **~42원** | **5.2%** |

## 🎯 이 분해가 말해주는 것 (지표만 봐선 절대 안 보임)
1. **봉지 1개 팔아 남는 건 딱 42원.** 5개 묶음(신라면 멀티팩) 팔아야 210원. 마진이 얇아 **물량(volume)이 전부**인 사업.
2. **포장재(봉지)가 밀가루·팜유만큼 큰 원가(~22%).** 그래서 2026년 포장재(BOPP 필름) 대란 때 "라면 팔수록 손해" 기사가 나옴 → **밀·팜유만 보면 절반만 본 것.**
3. **원재료(밀+팜유)가 출고가의 ~38%** → 소맥 -11%, 팜유 +9.8%처럼 엇갈리면 봉지당 마진이 수 원 단위로 출렁 → 그래서 판가 +7.2% 인상이 즉각 이익(+12.8%)으로 연결됨.
4. **제조노무비가 6%로 낮음** = 라인 자동화 완료. 인건비보다 **원자재·포장재·환율**이 마진 결정 변수.

## 🔧 재구성 방법론 (전 제품 확장용 템플릿)
```
STEP 1  소비자가 조사        (참가격/마트/편의점)
STEP 2  유통마진 차감 → 출고가 (업종별 표준 마진율)
STEP 3  출고가 × 공시 원가율  → COGS
STEP 4  COGS를 원재료 배분     (제품 중량 × 원자재 시세 + 업계 구조 추정)
STEP 5  판관비·영업이익 차감   → 봉지당 순마진
STEP 6  원자재 시세 변동 → 마진 민감도 (Δ팜유 10% → Δ봉지마진 ?원)
```
