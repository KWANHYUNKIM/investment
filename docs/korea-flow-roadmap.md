# 한국 경제 흐름 — 로드맵 (남은 작업)

국내 자금 흐름(부동산·국채·통화)을 보여주는 "한국 경제 흐름" 탭의 확장 계획.

## 완료 (키 불필요, 배포됨)
- 백엔드 `app/data/korea_flow.py` + `/api/data/korea-flow` 엔드포인트
- 프론트 `components/KoreaFlow.tsx` + "한국 경제 흐름" 탭
- 내용:
  - 부동산 자금 신호 — 리츠/부동산 ETF 7종(가격·등락·1주/1개월/3개월 수익률·52주高 대비)
  - 국채·채권 자금 신호 — 국고채3년·10년·종합채권 ETF
  - 자금 판정 — 리츠/채권 1개월 평균 수익률로 유입/이탈 자동 산출
  - 뉴스 동향 — 부동산 거래·집값 / 대출·규제 / 국채·통안채 발행 / 리츠 (RSS)
- 시세: FinanceDataReader, 뉴스: Google News RSS. **외부 키 없음.**

## 남은 작업 — 무료 API 키 필요

### 1. 한국은행 ECOS (env: `ECOS_API_KEY`) — 핵심 3종 완료
키 1개로 모든 통계표 조회. `ecos.py` + `/api/data/ecos-macro`, KoreaFlow 탭
"국내 돈 흐름 — 거시지표" 섹션(값·전년比·스파크라인). 12h 캐시.
- [x] **M2 통화량 + 증가율** — 161Y006/BBHA00(월). 최신 4,160조, 전년동월비 +5.7%
- [x] **가계신용 잔액 추이** — 151Y001/1000000(분기). 1,993조, 전년동기비 +3.5%
- [x] **주택매매가격지수** — 901Y113/H69A·R70A(월). 전국 종합, 전년동월비/MoM
- [x] **국고채 3년/10년 금리 + 장단기 스프레드** — 721Y001/5020000·5050000(월).
      최신 3년 3.68%·10년 4.08%, 스프레드 +0.4%p. KoreaFlow 거시지표 4번째 카드.
- [x] **통화안정증권(통안채) 발행잔액** — 191Y001(주요 국공채)/0800000(통안증권)·4(잔액),
      월. 최신 94.5조, 전년동월比 −9.7%. "국가(한은) 발행"의 실데이터. 거시지표 카드.
- [ ] **주택담보대출 잔액** — 181Y012는 '차주당'(평균)이라 총잔액 아님. 가계신용으로
      가계부채는 이미 커버. 총 주담대 잔액은 별도 표 탐색 필요(낮은 우선순위).
- 발급: https://ecos.bok.or.kr → OpenAPI → 인증키 신청(즉시·무료)
- 코드 찾는 법: StatisticTableList(표) → StatisticItemList(항목) → StatisticSearch(값)

### 2. 공공데이터포털 data.go.kr (env: `DATA_GO_KR_KEY`) — 일부 완료
- [x] **월별 아파트 실거래 거래대금·거래량** — `realestate.py` + `/api/data/realestate-trades`,
      KoreaFlow 탭 "부동산 실거래" 섹션. 최근 6개월, 당월=잠정 처리.
- [x] **전국 250개 시군구 확장** — `lawd_codes.py`(행정표준 법정동코드에서 리프 시군구
      추출, 구 있는 시는 구 단위). 시도별 거래대금 막대 + 상위 시군구. ~1,500콜/refresh,
      12h 캐시 + 스케줄러 워밍으로 첫 진입 대기 없음. 검증: 2026.05 전국 25.5조.
- [ ] (선택) **전월세 실거래·전세가율** 흐름 — "아파트 전월세 실거래가" API 추가 신청.
- [ ] 스케줄러 워밍 — 현재 첫 호출 ~15초(150콜). growth_scheduler에 6h 주기 워밍 추가 검토.
- 엔드포인트: `https://apis.data.go.kr/1613000/RTMSDataSvcAptTrade/getRTMSDataSvcAptTrade`
- 주의: 신규 키는 활용신청 후 게이트웨이 반영까지 수십 분 지연(403) 후 활성화됨.

> 키는 `backend/.env`에 추가 (기존 `DART_API_KEY`, `FINNHUB_API_KEY`와 동일 방식, config가 자동 로드).

### 3. 스케줄러 워밍 — 완료

      각자 TTL 존중(force 아님). 검증: 워밍 후 세 엔드포인트 <10ms.

### 4. 전세 — 완료
- [x] **주택전세가격지수**(전국·종합) — 901Y114/H69A·R70A(월). ECOS 6번째 카드.
- [x] **전월세 실거래** — `rent.py` + `/api/data/realestate-rent`. RTMSDataSvcAptRent로
      전국 250시군구 월별 거래량·전세/월세 비중·평균 전세보증금 + 시도별. 같은
      DATA_GO_KR_KEY로 동작. 검증: 2026.05 전국 67,067건·월세비중 51.2%·평균전세 3.82억.
- [ ] (선택) 전세가율 — 전세보증금/매매가 매칭 필요(동일 단지·면적), 난이도 높아 보류.
