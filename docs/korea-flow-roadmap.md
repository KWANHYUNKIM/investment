# 한국 경제 흐름 — 로드맵 (남은 작업)

국내 자금 흐름(부동산·국채·통화)을 보여주는 "한국 경제 흐름" 탭의 확장 계획.

## 완료 (키 불필요, 배포됨)
- 백엔드 `app/data/korea_flow.py` + `/api/data/korea-flow` 엔드포인트
- 프론트 `components/KoreaFlow.tsx` + "한국 경제 흐름" 탭
- 내용:
  - 부동산 자금 신호 — 리츠/부동산 ETF 7종(가격·등락·1주/1개월/3개월 수익률·52주高 대비)
  - 국채·채권 자금 신호 — 국고채3년·10년·종합채권 ETF
  - 자금 판정 — 리츠/채권 1개월 평균 수익률로 유입/이탈 자동 산출
  - 뉴스 동향 — 부동산 거래·집값 / 대출·규제 / 국채·통안채 발행 / 리츠 (RSS)
- 시세: FinanceDataReader, 뉴스: Google News RSS. **외부 키 없음.**

## 남은 작업 — 무료 API 키 필요

### 1. 한국은행 ECOS (env: `ECOS_API_KEY`)
키 1개로 모든 통계표 조회 가능. "국내 돈 흐름"의 핵심.
- [ ] **M2 통화량 + 증가율** — 시중 유동성이 느나/주나
- [ ] **가계신용·주택담보대출 잔액 추이** — 부동산으로 흘러든 빚
- [ ] **국고채 3년/10년 금리 + 장단기 스프레드** (fdr이 못 주는 수치)
- [ ] **통화안정증권(통안채) 발행잔액** — 한은의 시중자금 흡수/공급 ("국가 발행")
- [ ] **주택매매가격지수** — 실제 집값 방향
- 발급: https://ecos.bok.or.kr → OpenAPI → 인증키 신청(즉시·무료)

### 2. 공공데이터포털 data.go.kr (env: `DATA_GO_KR_KEY`)
- [ ] **월별 아파트 실거래 거래대금·거래량** (직접 요청받은 핵심 수치)
- [ ] **지역별(서울/수도권/지방) 거래 분포**
- [ ] (선택) **전월세 실거래·전세가율** 흐름
- 발급: https://www.data.go.kr → "아파트 매매 실거래가 (상세 자료)" 활용신청(자동승인)
- 주의: '상세 자료' 버전이라야 거래금액 포함

> 키는 `backend/.env`에 추가 (기존 `DART_API_KEY`, `FINNHUB_API_KEY`와 동일 방식, config가 자동 로드).

## 남은 작업 — 키 없이 추가 가능 (fdr)
- [ ] 자산 로테이션 ETF 추가 — 달러 ETF(KODEX 미국달러선물)·금 ETF(KODEX 골드선물): "원화 약세 → 달러·금 도피" 흐름
- [ ] 부동산 ETF 그룹 모멘텀 게이지(평균 수익률 한눈에)
- [ ] 개별 리츠 더 추가 (제이알글로벌리츠·신한알파리츠·코람코라이프인프라 등)

## 권장 순서
ECOS 키 → 거시지표(M2·국고채금리·통안증권) 연동이 "국내 돈 흐름"에 가장 임팩트 큼.
그다음 data.go.kr → 부동산 거래액.
